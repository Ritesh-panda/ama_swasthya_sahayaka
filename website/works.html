How It Works

From message to
life-saving guidance.

Five steps. Under 3 seconds. In your language.

01

User sends a message

A WhatsApp text, shared GPS location, or a spoken voice query — in any language, on any device. No app required.

HindiOdiaTeluguMarathiTamilBengali
📩
02

Language auto-detected and translated

Google Cloud Translation detects the user's native language and translates to English for uniform AI processing.

Google Translate API50+ languagesAuto-detect
🌍
03

Gemini AI classifies intent

Gemini 2.5 Flash reads the message with a strict system prompt and outputs a structured JSON: intent, extracted data, confidence score.

EMERGENCYSYMPTOMVACCINEHOSPITALGENERAL
🧠
04

Central orchestrator routes to the right service

process_query.py calls the right backend service — Google Places API for hospitals, PostgreSQL for vaccines, or live AI for symptoms.

Google Places APIPostgreSQL + PostGISNewsAPI
⚡
05

Response delivered in user's language

The reply is translated back, then formatted for the channel — rich WhatsApp markdown with emojis, or clean short sentences for voice.

TwiML XML (WhatsApp)JSON (Voice)Translated reply
✅

Channel-specific flows

Same brain, different delivery.

💬 WhatsApp Channel
  • 1
    Twilio receives the WhatsApp message and POSTs to /api/v1/chat
  • 2
    Language detected → translated to English
  • 3
    process_query() orchestrator called with source="whatsapp"
  • 4
    Response formatted with emojis + markdown for WhatsApp display
  • 5
    Translated back → returned as TwiML XML to Twilio
🎙️ Voice Channel
  • 1
    Browser detects location → maps to nearest Indian state
  • 2
    Firebase Auth → anonymous sign-in → load saved preferences
  • 3
    Gemini Live Audio streams real-time microphone audio
  • 4
    Safety guardrails evaluated in real-time (no diagnosis, no prescriptions)
  • 5
    Gemini responds with voice audio in the user's detected language
