🎙️ Voice App 💬 WhatsApp
Technology

World-class
infrastructure.

Built on Google Cloud, FastAPI, and Gemini AI — production-grade from day one.

🐍

Backend Core

  • FastAPI High-performance Python ASGI framework
  • PostgreSQL Relational DB with PostGIS geospatial extension
  • SQLAlchemy ORM + Alembic migrations
  • Redis Session history cache (1hr TTL)
  • CORS JSON API bridge for voice frontend
🧠

AI & Intelligence

  • Gemini 2.5 Flash Intent classification + triage
  • Gemini Live Audio Real-time voice streaming
  • Google Translate 50+ language auto-detection
  • Google Places API Hospital geosearch (10km radius)
  • Structured JSON Forced schema output from Gemini
📡

Channels & Delivery

  • Twilio WhatsApp Messaging API + Webhooks
  • Firebase Auth Anonymous sign-in for voice users
  • Firestore Persistent preference storage
  • React + Vite Voice frontend (TypeScript)
  • ngrok / Render Tunnel + cloud deployment
🗄️

Data Architecture

  • Users Phone number → preference mapping
  • Hospitals PostGIS POINT geometry for spatial search
  • Doctors Specialty indexed, FK to hospital
  • Vaccines WHO schedule by week range
  • Appointments User × Doctor × Hospital × Time

Safety by Design

Every response passes through medical safety guardrails.

  • 🚫 No confident medical diagnosis
  • 🚫 No prescription or dosage changes
  • 🚫 No free-form lab interpretation
  • ✅ Emergency → 108 escalation first
  • ✅ Advisory guidance only
  • ✅ No personal health data retained
